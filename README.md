# n8n Workflow Collection

This repository contains a collection of JSON workflow files for n8n, a workflow automation tool. These workflows are designed to automate various tasks and processes.

## Categories

### AI Agents & Chatbots
*   **MSDAT Main Chat Agent**: Main conversational agent workflow.
*   **MSDAT Sub Agent**: Sub-agent workflow for specific tasks.
*   **e4e Slack Moderator**: Automation for Slack moderation tasks.

### Scheduling & Operations
*   **CEO Meeting Schedule**: Workflow for managing meeting schedules.
*   **NHMIS Annual Update**: Automation for the National Health Management Information System annual update process.

### Task & Knowledge Management
*   **Clickup Task Creation**: Automates the creation of tasks in ClickUp.
*   **Task Knowledge base**: Manages knowledge base entries related to tasks.

## Usage
To use these workflows, import the JSON files into your n8n instance.
